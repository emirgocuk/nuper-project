# Product Context

## Why this project exists
Nuper exists to create a meaningful, high-value connection point between entrepreneurs and investors. The "Elite" version aims to elevate this interaction from a standard directory to a prestigious, trusted platform.

## Problems it solves
- **Lack of Trust:** Solved by Verification Badges (Verified Investor, MVP Entrepreneur profiles).
- **Investor Spam & Fatigue:** Solved by the "Double Opt-in" (Tinder for Startups) messaging system, where both parties must accept the connection before speaking.
- **Information Overload:** Solved by the "Data Room" concept. Only the 1-minute "Elevator Pitch" video is public; financial details and pitch decks are exclusively locked for verified, granted investors.
- **Generic Experiences:** Solved by personalized onboarding and role-based matchmaking (Nuper Match Feed).
- **Investment Risk:** Solved by "Syndicate" pools, allowing smaller investors to co-invest alongside Lead Investors.

## How it should work
- **Video-First Onboarding:** Entrepreneurs create a 1-minute elevator pitch video during the visually engaging onboarding process.
- **Showcase Feed:** The platform provides a customized, infinite-scroll shorts/reels feed of project pitches.
- **The Data Room:** Interested investors request access to the Data Room for deep-dive documents.
- **Match & Speak:** If the investor likes the project and the entrepreneur accepts, a secure messaging channel opens.
- **Public Profiles (`nuper.com/username`):** Showcase "Nuper Profile Cards" with radar charts of skills and timelines of achievements.

## User experience goals
- **Premium Aesthetic:** Glassmorphism, deep space themes, dark blue with neon accents.
- **Dynamic Interaction:** Scroll effects, 3D tilt cards, magnetic buttons, and fade animations seamlessly integrated using Framer Motion.
- **High-End Typography:** Space Grotesk and Inter to convey a modern, elite tech brand.
