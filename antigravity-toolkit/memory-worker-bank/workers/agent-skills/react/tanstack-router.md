# When to Use This Skill
- •Building complex React SPAs requiring advanced routing and nested layouts.
- •Implementing server-side rendering (SSR) or static site generation (SSG) with robust data loading.
- •Developing applications that benefit from type-safe navigation and URL parameter validation.
- •Integrating routing with data fetching libraries like TanStack Query for optimized performance.

# Pro Tips
- 💡Always define your `createRootRoute()` with an `<Outlet />` in `__root.tsx` to ensure proper rendering of child routes.
- 💡Leverage the Zod adapter for robust URL parameter and search parameter validation, enhancing type safety and preventing runtime errors.
- 💡Organize your route files logically within the `src/routes/` directory to fully utilize the file-based routing auto-generation and maintain clarity.
