# audit-context-building
The Audit Context Building skill transforms how an AI agent approaches code analysis, focusing intensely on foundational understanding. Instead of immediately hunting for issues, it methodically constructs a comprehensive mental model of the codebase. By analyzing line-by-line and linking insights from micro to macro, this skill ensures a deep, stable comprehension. It's designed to precede any discovery phase, significantly reducing errors and enhancing the quality of subsequent findings by grounding them in undeniable architectural truth.
