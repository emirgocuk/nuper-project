Navigating the complex landscape of blockchain development demands meticulous verification to bridge the gap between abstract design and concrete implementation. The Spec-to-Code Compliance Agent Skill provides a critical lens, acting as an expert auditor to ensure every line of code aligns perfectly with your project's architectural blueprints and whitepapers. It's an essential tool for maintaining integrity, mitigating risks, and building trust in decentralized systems, offering an unparalleled level of scrutiny for even the most intricate protocol implementations.

# When to Use This Skill
- •Auditing new smart contract deployments against official whitepapers.
- •Identifying discrepancies between a blockchain protocol's written specification and its codebase.
- •Performing regulatory compliance checks for decentralized finance (DeFi) applications.
- •Validating an existing blockchain project's implementation against updated design documentation.

# Pro Tips
- 💡Always provide the most up-to-date and complete versions of both the specification documents and the codebase for the most accurate analysis.
- 💡For large projects, break down the analysis into modules or specific features to get more granular and actionable compliance reports.
- 💡Pair this skill with `audit-context-building` to first establish a comprehensive understanding of the project before deep-diving into compliance checks.
