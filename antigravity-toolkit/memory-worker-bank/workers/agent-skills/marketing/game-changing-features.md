Unlock exponential growth for your product with the 'Game-Changing Features' Agent Skill. This powerful tool acts as your personal product strategist, pushing beyond incremental improvements to identify truly transformative opportunities. It challenges you to think like a founder, focusing on moves that multiply user value by ten, not just add marginal gains. Leverage this skill to uncover the core differentiators that will make your offering indispensable to users, guiding your development towards truly impactful innovation.

# When to Use This Skill
- •When you need strategic product thinking to identify your next big move.
- •When the user explicitly mentions '10x' or wants to make the product '10x better'.
- •To brainstorm high-impact features that would redefine the product's value proposition.
- •When you're asking 'what should we build next' from a strategic, long-term perspective.

# Pro Tips
- 💡Provide a clear, concise overview of your product's current state and target audience to ensure focused strategic output.
- 💡Explicitly state any hard constraints (e.g., technical, budget, timeline) at the outset to guide the brainstorming within realistic bounds.
- 💡Use the output as a foundation for further user research and competitive analysis, validating the '10x' ideas before committing to development.
