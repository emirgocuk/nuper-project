Many struggle to translate theoretical knowledge from articles or videos into practical application. This AI Agent Skill bridges that gap, empowering users to actively transform any educational content into tangible, actionable steps. By following the 'Ship-Learn-Next' framework, you can move beyond mere consumption to consistent iteration and real-world skill development. It's designed for those who want to accelerate their learning curve, ensuring every piece of information leads to a concrete action, measurable progress, and continuous improvement in their coding or domain expertise.

# When to Use This Skill
- •After watching a coding tutorial, generate a step-by-step project plan to build a similar app from scratch.
- •Convert a long-form technical article into a series of practical exercises or 'reps' to master a new concept or tool.
- •Transform a YouTube transcript on a new framework into a phased learning quest, complete with coding challenges and checkpoints.
- •Break down complex architectural advice from a mentor into shippable tasks for an ongoing software development project.

# Pro Tips
- 💡Provide the learning content in a clean, easily parsable format (e.g., plain text, well-structured markdown for transcripts/articles) for optimal extraction.
- 💡Clearly define your desired 'ship' outcome upfront to guide the skill's output towards a specific, measurable goal.
- 💡After receiving the 'next' steps, commit to the first 'ship' phase quickly, then use the 'learn' phase to reflect and refine subsequent iterations with the skill.
