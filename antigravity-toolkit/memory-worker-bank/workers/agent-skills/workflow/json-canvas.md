This skill elevates your AI agent's capability to interact with the innovative JSON Canvas format. Designed for seamless integration with tools like Obsidian, it enables the creation, manipulation, and organization of complex visual data structures. Whether you're mapping out ideas, detailing project workflows, or structuring knowledge, the skill provides a programmatic interface to manage nodes, edges, and groups. It ensures data integrity and adherence to the JSON Canvas Spec, allowing your agent to generate highly structured and visually intuitive canvases that enhance productivity and understanding.

# When to Use This Skill
- •Generating a new mind map from a block of text, automatically creating nodes and connections.
- •Refactoring an existing .canvas file to add new nodes, modify content, or re-group elements based on user instructions.
- •Creating a detailed flowchart for a software process, including file references and external links as distinct node types.
- •Programmatically populating a visual knowledge base in Obsidian Canvas from structured data sources or research notes.

# Pro Tips
- 💡Leverage the `group` node type to logically organize related information, significantly improving the readability and maintainability of complex canvases.
- 💡When creating new nodes, specify the `id` attribute carefully to ensure uniqueness across the canvas, as it's crucial for establishing valid edge connections.
- 💡Always validate the generated JSON against the official JSON Canvas Spec 1.0 to prevent parsing errors and ensure compatibility with target applications like Obsidian.
