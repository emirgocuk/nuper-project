-- Monthly Recurring Revenue (MRR)WITH mrr_calculation AS (    SELECT        DATE_TRUNC('month', billing_date) AS month,        SUM(            CASE subscription_interval                WHEN 'monthly' THEN amount                WHEN 'yearly' THEN amount / 12                WHEN 'quarterly' THEN amount / 3            END        ) AS mrr    FROM subscriptions    WHERE status = 'active'    GROUP BY DATE_TRUNC('month', billing_date))SELECT    month,    mrr,    LAG(mrr) OVER (ORDER BY month) AS prev_mrr,    (mrr - LAG(mrr) OVER (ORDER BY month)) / LAG(mrr) OVER (ORDER BY month) * 100 AS growth_pctFROM mrr_calculation;-- Cohort RetentionWITH cohorts AS (    SELECT        user_id,        DATE_TRUNC('month', created_at) AS cohort_month    FROM users),activity AS (    SELECT        user_id,        DATE_TRUNC('month', event_date) AS activity_month    FROM user_events    WHERE event_type = 'active_session')SELECT    c.cohort_month,    EXTRACT(MONTH FROM age(a.activity_month, c.cohort_month)) AS months_since_signup,    COUNT(DISTINCT a.user_id) AS active_users,    COUNT(DISTINCT a.user_id)::FLOAT / COUNT(DISTINCT c.user_id) * 100 AS retention_rateFROM cohorts cLEFT JOIN activity a ON c.user_id = a.user_id    AND a.activity_month >= c.cohort_monthGROUP BY c.cohort_month, EXTRACT(MONTH FROM age(a.activity_month, c.cohort_month))ORDER BY c.cohort_month, months_since_signup;-- Customer Acquisition Cost (CAC)SELECT    DATE_TRUNC('month', acquired_date) AS month,    SUM(marketing_spend) / NULLIF(COUNT(new_customers), 0) AS cac,    SUM(marketing_spend) AS total_spend,    COUNT(new_customers) AS customers_acquiredFROM (    SELECT        DATE_TRUNC('month', u.created_at) AS acquired_date,        u.id AS new_customers,        m.spend AS marketing_spend    FROM users u    JOIN marketing_spend m ON DATE_TRUNC('month', u.created_at) = m.month    WHERE u.source = 'marketing') acquisitionGROUP BY DATE_TRUNC('month', acquired_date);

This skill empowers AI agents to guide users through the intricate process of creating impactful Key Performance Indicator (KPI) dashboards. Beyond just charting data, it focuses on strategic metric selection, ensuring every visualization aligns with business objectives. From setting up real-time monitoring to establishing clear data governance, this expertise enables the creation of analytical tools that genuinely drive decision-making. Leverage its guidance to transform raw data into actionable insights, making your dashboards not just informative, but truly transformative for any organization. This skill bridges the gap between data and decisive action.

# When to Use This Skill
- •Developing a new executive summary dashboard from scratch
- •Optimizing an existing sales performance dashboard for better clarity and impact
- •Setting up real-time operational dashboards for manufacturing or IT monitoring
- •Defining a hierarchy of KPIs for different organizational levels (strategic, tactical, operational)

# Pro Tips
- 💡Always start with the 'why' – what key decision should this dashboard enable or improve?
- 💡Prioritize clarity over complexity; a few impactful visuals are better than many confusing ones.
- 💡Iterate with stakeholders: user feedback is crucial for dashboard adoption and effectiveness.
- 💡Ensure your data sources are reliable and consistently updated for accurate KPI reporting.
