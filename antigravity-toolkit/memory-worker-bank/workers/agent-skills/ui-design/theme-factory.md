The Theme Factory skill empowers AI agents to effortlessly manage the visual consistency of various digital artifacts. It provides a robust framework for applying professional design principles, ensuring all output, from detailed reports to engaging slide decks, adheres to a unified aesthetic. This capability significantly elevates the quality of AI-generated content by embedding a strong sense of brand identity or thematic coherence, making interactions with AI-produced materials more professional and impactful without manual intervention.

# When to Use This Skill
- •Creating visually consistent slide decks for business presentations or academic lectures.
- •Styling automatically generated reports or technical documentation to match corporate branding.
- •Producing themed HTML landing pages or web components with a cohesive look and feel.
- •Ensuring brand consistency across multiple types of AI-generated content and artifacts.

# Pro Tips
- 💡Always display the `theme-showcase.pdf` first to allow users to visually select a theme, improving user experience and choice accuracy.
- 💡Integrate theme selection early in content generation workflows to ensure all subsequent elements are styled correctly from the outset.
- 💡Consider automating theme application based on the context of the artifact (e.g., 'corporate theme' for reports, 'modern theme' for marketing materials).
