# Pro Tips
- 💡Always start with normalization to 3NF and only denormalize if specific performance bottlenecks are identified and proven through testing.
- 💡Prioritize indexing on foreign key columns and frequently queried WHERE clause columns; avoid over-indexing, which can hurt write performance.
- 💡Regularly review `EXPLAIN ANALYZE` output for your most critical queries to validate your design choices and identify further optimization opportunities.
