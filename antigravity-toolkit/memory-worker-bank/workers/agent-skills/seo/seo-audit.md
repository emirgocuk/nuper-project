# seo-audit
Leverage the power of AI to gain a decisive edge in search engine optimization. This specialized agent skill acts as your personal SEO consultant, meticulously analyzing your website for critical issues that hinder visibility and performance. Beyond just identifying problems, it provides a structured framework and practical, step-by-step recommendations. Whether you're a seasoned marketer or a new site owner, this tool empowers you to transform complex SEO challenges into clear, executable tasks, driving measurable improvements in organic traffic and search rankings.

# When to Use This Skill
- •Diagnosing a sudden drop in organic search traffic or rankings.
- •Performing a comprehensive technical and on-page SEO health check for a new website launch.
- •Identifying specific optimization opportunities for underperforming pages or content.
- •Reviewing a site's meta tags, sitemap, and robots.txt file for critical errors.

# Pro Tips
- 💡Always provide the agent with as much context as possible, including site type, business goals, and access to any relevant data (e.g., Search Console insights) for more precise recommendations.
- 💡Clearly define the scope of the audit (e.g., 'focus on crawlability issues only' or 'audit blog posts published in the last quarter') to get targeted results.
- 💡Prioritize the AI's recommendations by potential impact and ease of implementation, starting with critical technical fixes before moving to on-page refinements.
