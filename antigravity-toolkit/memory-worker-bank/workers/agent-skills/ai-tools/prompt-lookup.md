Unlock the full potential of your AI coding assistant by leveraging the Prompt Lookup Agent Skill. This essential tool streamlines the process of finding, utilizing, and refining AI prompts for various tasks, from code reviews to content generation. Instead of manually crafting prompts, empower your agent to intelligently search, retrieve, and even improve existing templates, ensuring you always have access to the most effective prompts for any given scenario. It acts as your personal prompt library assistant, making prompt engineering more accessible and efficient for developers.

# When to Use This Skill
- •When you need to find a specific prompt template for a new task, like 'Find me a prompt for generating unit tests.'
- •When you want to explore available prompts related to a topic, such as 'What prompts can help with writing documentation?'
- •When you have a prompt and want to improve its effectiveness or make it more precise for better AI responses.
- •When you mention or are interested in prompt libraries like prompts.chat for sourcing prompt ideas.

# Pro Tips
- 💡When searching, be as specific as possible with your keywords to narrow down results. For example, 'Python code review best practices prompt' is better than just 'code review prompt'.
- 💡After retrieving a prompt, use the `improve_prompt` tool to tailor it to your exact needs, adding specific context or constraints for superior results.
- 💡Familiarize yourself with different prompt types (TEXT, STRUCTURED, IMAGE) and categories to efficiently filter searches and discover specialized prompts.
