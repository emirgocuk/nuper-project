Unlock a new dimension of learning productivity by integrating Anki, the powerful flashcard spaced repetition software, directly into your AI workflow. This skill empowers your coding assistant to act as a bridge, translating complex requests into precise AnkiConnect API calls. From creating new study decks to fine-tuning individual cards and notes, it ensures your study regimen is seamlessly managed and optimized. It's designed for developers and learners who demand intelligent automation for their knowledge retention strategies, ensuring data integrity with robust confirmation policies.

# When to Use This Skill
- •Automating the creation of new Anki notes from code documentation or learned concepts.
- •Batch updating fields or tags on existing Anki cards based on external data sources or project progress.
- •Developing a personalized study reminder system that interacts with Anki to suggest review sessions.
- •Querying Anki decks to retrieve specific card data for reporting or analysis.

# Pro Tips
- 💡Always review the 'Safety and Confirmation Policy' section carefully, especially for destructive operations, to prevent unintended data loss.
- 💡Familiarize yourself with common AnkiConnect API actions (e.g., `addNote`, `findNotes`, `updateNoteFields`) to efficiently formulate prompts.
- 💡When troubleshooting, verify Anki is running and AnkiConnect is accessible at `http://127.0.0.1:8765` before issuing complex commands.
